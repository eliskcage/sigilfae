# SIGILFAE TESSERA

> Satoshi-QR-triangle fairy. Token: `t`. Gold.
> The QR is the password itself.

A fairy whose chest is a glowing **Satoshi-QR triangle**. TESSERA is the
empire's password-AND-page primitive. When clicked, she opens a gate that
demands the user type a unique **QR-string** — knowing the string is having
access. The string serves as identifier, password, and page-builder all at
once.

## Behaviour

- **Click TESSERA (or any APIS bee)** → opens the QR-gate modal.
- User types their unique string into the input.
- Valid string → redirects to `base_page.html?qr=<string>` — the user's
  personal **base of operations**.
- New users can mint a fresh TESSERA from the modal.

## What's a base of operations?

A personal page built from the godmesh primitives. Contains:

- **Live fairy graph** — the user's own DOTGRAMMA (iframed from `fairy_coder.html`)
- **YouTube prototype embed** — a video showing the idea
- **mcForm element** — empire's existing form-builder, embedded
- **QR pins** — links to other base-pages (each pin = a related idea)
- **Vote buttons** — works / shit (audience verdict)
- **Share URL** — anyone with the URL can visit; only QR-holders can edit

The empire's idea-network = a graph of these inter-linked base-pages.

## Why "Tessera"?

Latin for **tile** or **token of identity** — a Roman tessera was a small
mosaic-tile or a clay ID-token used as proof-of-membership in clubs, the
army, the senate. Empire-flavoured + etymologically aligned with NOTAE,
PYXIS, CRUX, ANIMA, PANIS, OPERA, APIS.
