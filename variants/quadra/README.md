# SIGILFAE QUADRA

> Square-QR fairy. Token: `Q`. Gold.
> The empire's signature shape · distinguishes a SIGILFAE base from a shortfactory kingdom-piece.

A fairy whose chest is a glowing **square Satoshi-QR** (vs TESSERA's
triangle). QUADRA is the empire's signature gate — used wherever a
SIGILFAE-system base-page should be visually distinct from a
shortfactory.shop kingdom-piece (which uses the triangle pattern).

## Why a square?

The shortfactory kingdom-piece system at `shortfactory.shop/q/<hash>` uses
the triangular Satoshi-QR (TESSERA) for its existing personal-kingdom
pages. The SIGILFAE empire is a **parallel surface**, not a duplicate —
so it needs its own distinct shape.

Triangle = kingdom (existing).
Square = SIGILFAE empire (this).

A user can hold both: a triangular kingdom-piece on shortfactory AND a
square SIGILFAE base. Same soul, two surfaces, distinct visuals.

## Behaviour

Identical to TESSERA: click → opens the QR-gate modal → user types their
QR-string → if valid, redirects to `base_page.html?qr=<string>`.

The only difference is the chest icon (square vs triangle) and which
surface they represent.

## Why "Quadra"?

Latin for **square** (or four-sided figure). Empire-flavoured + 
etymologically aligned with NOTAE, PYXIS, CRUX, ANIMA, PANIS, OPERA, APIS,
TESSERA.
