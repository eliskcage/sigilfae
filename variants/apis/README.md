# SIGILFAE APIS

> Angry-bee quid-pro-quo fairy. Token: `q`. Yellow + black stripes, red stinger.

A fairy with a striped bee body and a red stinger pointing down. APIS
declares: **"I want X from the empire in exchange for declaring this
Idea Factory system."** Hostile-but-honest. Won't activate the surface
she's on until the empire negotiates / accepts the trade.

She shakes when rendered — angry-shake animation, ready to sting.

## Behaviour

- APIS is a **pre-condition fairy** — her presence in a mesh signals an
  unmet condition. Until her quid-pro-quo is resolved (per upstream UI),
  the rest of the mesh is blocked.
- Click → opens the negotiation modal (stub: shows a toast saying
  "APIS demands compensation").
- POP only after the trade is resolved.

## DOTGRAMMA usage

```
q                 → 1 angry bee, alone on stage
q-q-q             → 3 angry bees in triangle (multiple unmet conditions)
.^q               → a node + an APIS demanding something for that node
```

## Why "Apis"?

Latin for "bee". Empire-flavoured + etymologically on-theme with the
other binomials.
