# SIGILFAE PANIS

> Biscuit fairy. Token: `b`. Empire's affirmation token.

A cream-bodied fairy with a small biscuit on her chest. Empire's
**affirmation / thanks / small-reward** primitive. Used to record
acknowledgement, witness, or small-positive feedback.

## Behaviour

Decorative + ceremonial. Click cycles state; POP grants standard white blood.
PANIS is most useful as a leaf in tree-mode meshes — any time a "small win"
needs to be marked, drop a `b` in the bottom row.

## DOTGRAMMA usage

```
b                 → 1 biscuit fairy
b-b-b-b           → 4 biscuits in a closed-square formation
b^.               → biscuit + node under a shared parent
.^b .^b .^b       → 6-leaf tree mixing nodes and biscuits → cascade up
```
