# SIGILFAE ANIMA

> Soul-bond fairy. Token: `s`. Indigo.

A fairy whose chest displays the user's **SFT soul-file QR**. By appearing
in a DOTGRAMMA string, ANIMA declares that the bearer's blood-connection to
their account is LIVE and active on the visualizer — every subsequent action
mediated by this mesh is signed against that soul.

## Behaviour

- Click → cycles state (open → in-progress → complete → POP).
- POP grants white blood **to the soul whose QR is on the chest**, not the
  generic active soul. ANIMA is therefore the binding primitive between any
  fairy mesh and a real account.

## DOTGRAMMA usage

```
s             → 1 ANIMA, declaring soul-bond
s-s-s         → 3 ANIMAS in a line, three souls bonded into one mesh
s^j           → ANIMA + jobs-list under a shared parent (a soul taking on a job-list)
```
