# SIGILFAE NOTAE

> The base species. Note-fairies. Token: `.`

The canonical Sigilfay. A small cream-bodied fairy with eyes-only face,
translucent dragonfly wings, and an amber pentagon glowing on her chest.

## Behaviour

Decorative node. Click cycles state (open → in-progress → complete → POP).
Drag moves freely. POP grants white blood.

## Why pentagon?

The pentagon is the empire's chest-glow primitive across all character work
(see `claude_fairy.js`). Five-sided = five empire principles: covenant,
ritual, repair, witness, joy.
