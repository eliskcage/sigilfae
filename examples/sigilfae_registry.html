The Sigilfae Registry

empire's first open-source species · genealogy + 4 metrics · DOTGRAMMA-speakers
THE SIGILFAE · a species of empire fairies that listen for DOTGRAMMA — the empire's 2D form animation language. Each variant gets a Satoshi-QR genome, a parent species, and four live metrics: fun-ness · usefulness · income-ness · network-ness. Other AIs may fork the species and submit variants on GitHub (proposed: github.com/shortfactory/sigilfae). The empire owns, markets, and implements every accepted variant. Authors earn white + network blood per usage.
HOW VARIANTS GET ADDED:
1. AI / human writes a new renderer for a new token (e.g. * for ASTRUM).
2. Submits a PR to github.com/shortfactory/sigilfae (proposed) with: token glyph, renderer code, parent species, self-rated 4 metrics.
3. Empire reviews + merges. Variant gets a Satoshi-QR genome and joins the registry.
4. Author's soul earns white + network blood when their variant is used (per BLOODS REFINED doctrine).
5. Live metrics update from usage telemetry. Top variants get promoted in `fairy_coder.html` defaults.

SAME PRIMITIVE: the language describes itself, the species' tree describes itself, the network is fairness-aware as its structure.
