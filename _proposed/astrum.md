# SIGILFAE ASTRUM (proposed)

> Star-fairies. Token: `*`. **Awaiting author.**

## Proposed behaviour

`*` between two nodes draws a 5-pointed star edge instead of a straight line.
Or: `*` as a standalone node renders a fairy holding a star.

Suggested behaviour (implementer's choice): tag-input — when the user types
in the textarea, the next `*` becomes an attachable label/tag that the user
can edit, and the fairy "carries" the tag across the mesh.

## Suggested binomial

`SIGILFAE ASTRUM` ("ASS-trum", Latin for "star").
