# SIGILFAE CIRCULUS (proposed)

> Circle-fairies / orbit-nodes. Token: `o`. **Awaiting author.**

## Proposed behaviour

Each `o` node renders as a fairy that ORBITS its polygon vertex rather than
staying fixed. Multiple `o` tokens create a system of orbiting nodes.

Suggested input behaviour: radio-toggle — clicking a CIRCULUS fairy toggles
between two states (on/off) rather than the standard 4-state cycle.

## DOTGRAMMA examples

```
o-o-o            → 3 orbiting nodes
o-.-o            → 1 orbiting + 1 NOTAE + 1 orbiting
```

## Suggested binomial

`SIGILFAE CIRCULUS` ("KIR-koo-lus", Latin for "small circle").
