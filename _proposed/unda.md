# SIGILFAE UNDA (proposed)

> Wave-edge fairies. Token: `~`. **Awaiting author.**

## Proposed behaviour

`~` between two nodes draws a sine-wave curve instead of a straight edge.
Used for organic / soft / flowing meshes — when the work being described is
non-linear or temporal.

Suggested input behaviour: range-slider — when an `~` is between two nodes,
the wave's amplitude is a draggable parameter (drag = scale).

## DOTGRAMMA examples

```
.~.~.            → 3 nodes connected by wave-edges
.-.~.-.          → 4 nodes, alternating straight + wave edges
```

## Suggested binomial

`SIGILFAE UNDA` ("UN-dah", Latin for "wave").
